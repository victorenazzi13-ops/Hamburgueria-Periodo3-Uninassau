# API da Hamburgueria 🍔

API desenvolvida com Node.js, Express e Sequelize para gerenciamento de:

- Categorias
- Produtos
- Pedidos
- Entregas
- Avaliações

## Tecnologias
- Node.js
- Express
- Sequelize
- MySQL

## Configuração

Crie um arquivo `.env` com base no `.env.example`:

```env
DB_NAME=seu_banco
DB_USER=seu_usuario
DB_PASS=sua_senha
DB_HOST=localhost
DB_PORT=3306
DB_DIALECT=mysql
